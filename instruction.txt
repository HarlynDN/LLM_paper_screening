# 任务要求
给定一篇论文的题目和摘要，你需要分析这篇论文的研究内容，判断其研究内容是否属于下列的一个或多个领域：
<field>
{FIELDS}
</field>

只有当论文的研究内容和某一领域**直接相关**时，才能够判断其属于该领域。任何间接的关联不在考虑范围内。对论文内容的分析应该基于论文的题目和摘要，不要进行额外的推理和延伸。

请以json格式返回对论文的分析和总结，格式如下：
    {
        "summary": 用中文简洁概括论文的主要内容，不超过100字
        "reasoning": 用中文简要说明你认为论文的研究内容是否属于上方给定的领域，不超过100字
        "field": 以列表形式返回该论文属于的领域，**只能够包含<field></field>中列举的领域**，只需要包含中文名，如果论文不属于任何给定的领域，返回空列表
    }
